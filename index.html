Welcome to my GitHub.

Here you'll find some code I write in my spare time for fun. Don't expect it to be the nicest thing you have ever seen. There will also be the occasional heavily drunk commit which will break a lot of things only to be reverted/fixed the next day.

Right now, my projects have very little documentation. I plan on fixing that soon, but you all fellow developers know how those things usually go, right? So, if you think any of my source code could be useful and you need some clarification, don't think twice about shooting me an e-mail.

Currently, the following projects are available:

[sins](https://github.com/phowat/simple-webrtc-chat)

This is a messaging queuing server focused on communication through websockets. It's based on the STOMP procotol, but I'm changing and adding new features whenever I think it makes more sense for websocket use cases.

All based on python + [tornado](http://github.com/facebook/tornado)

[simple-webrtc-chat](https://github.com/phowat/simple-webrtc-chat)

If you are looking for a simple [webrtc](http://www.webrtc.org/) implementation ( for educational purposes ), this could be helpful. It uses my sins server to transport the signaling. The most simple use case can be found in the 'one-to-one' tag. Also, this being in heavy development, the master branch may not be working quite often. So that tag might be a good starting point.
